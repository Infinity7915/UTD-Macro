## 🌟 **What is Project Infinity🪽?**

Project Infinity🪽 is a **smart Lua macro** for Roblox’s *Ultimate Tower Defense*.  
It handles everything your fingers don’t want to:

- 🧠 Remembers your tower placements (even sub‑towers)
- 💰 Waits until you have enough cash before placing
- 🖱️ Auto‑clicks placement & upgrade buttons with perfect timing
- 🎯 Automatically returns to lobby & re‑enters matches
- 📂 Saves your data locally (no re‑recording every session)

> **“Set it once, farm forever.”**

## 🧰 **How to Use**

1. **🔧 Load the script** place the script in your executors auto-execute folder.
2. **📋 Open the UI if closed** (default key: `K`).
3. **🧐 ↓ Classic Match Set-up**
4. **🎯 Record Tower CFrames**:
   - Join any game mode
   - Stand exactly where you want a tower placed.
   - Click `Record Cframe` – the script saves your exact position.
   - Enable `Sub‑Tower` to add multiple spots per tower (if you can place 1 tower mu).
5. **💰 Set Place Costs**:
   - Select a tower in the dropdown.
   - Enter its cost (e.g., `1500`) and hit `Set Cost`.
   - The macro will only place the tower when you have enough cash.
6. **⚙️ Configure & Start**:
   - Choose `Easy` or `Hard` mode.
   - Toggle the **Macro ON**.
7. **💤 Sit back** – the script will walk you to the portal, start a match, and place/upgrade everything automatically.  
   It will also return to lobby and re‑queue when the round ends.
